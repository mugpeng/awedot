# awesession — 需要修改的点

> 项目路径: `product/tools/archive/awesession`

## #19 移除 tokio 依赖

`Cargo.toml:19` — 代码中没有任何 `tokio::` 直接使用，Tauri v2 自带 async runtime。

```toml
# 移除此行
tokio = { version = "1", features = ["full"] }
```

## #1 原子文件写入

`src-tauri/src/bookmarks/store.rs:58` — `fs::write` 非原子，崩溃可能丢数据。

```rust
// 当前
fs::write(&path, json).map_err(|e| ...)?;

// 改为 tmp + rename
let tmp = path.with_extension("json.tmp");
fs::write(&tmp, &json).map_err(|e| ...)?;
fs::rename(&tmp, &path).map_err(|e| ...)?;
```

## #6 started_at 字段语义错误

`src-tauri/src/sessions/discovery.rs:90-91` — `started_at` 和 `last_activity` 用了同一个 mtime。

```rust
// 当前
started_at: mtime.clone(),
last_activity: mtime,
```

两种修法：从 JSONL 第一行提取真实创建时间，或将 `started_at` 重命名为 `first_seen_at`。
